# ST LIS2DW12 3-axis motion Sensor Package

ST LIS2DW12 3-axis motion Sensor I2C extension for makecode.  

Author: shaoziyang  
Date:   2019.Jul  

## Basic usage
```
basic.forever(function () {
    serial.writeNumbers(LIS2DW12.get())
    basic.pause(500)
})
```
